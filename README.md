🎲 Dice Roll Game (Python)

🔹 Overview
An interactive **Dice Roll Game** built using Python.
The game allows a user to play against the computer over multiple rounds with a styled **CLI interface**, dice visuals, and score tracking.


🚀 Features
* 🎯 Multi-round gameplay
* 🎲 Random dice roll (1–6) for computer
* 🎨 Styled dice visuals using ASCII
* 📊 Scoreboard with final result
* 🧩 Modular code structure


🛠️ Tech Stack
* Python
* Random Module
* Modular Programming


📂 Project Structure
dice-roll-game/
│── main.py
│── ui.py
│── game_logic.py
│── scoreboard.py
│── utils.py


▶️ How to Run
python main.py


📌 Key Concepts Learned
* Random number generation
* Modular code design
* CLI formatting and user interaction
* Game logic implementation
